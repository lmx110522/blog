XML

可扩展标记语言，是一种标记语言
非常灵活，没有固定的标签，所有的标签都是自定义！通常用于信息的记录和传递！

xml格式

格式良好的xml文档

  • 必须有xml声明语句
  • 必须有且仅有一个根元素
  • 标签大小写敏感
  • 属性值用双引用
  • 标签成对
  • 元素正确嵌套

有效的XML文档

  • 首先必须有良好的格式
  • 使用DTD和XSD定义语义约束
